# Inputs (WIP)

## Required

The `input` element should have a visible focus indicator.

```css
.input:focus-visible {
  outline-color: var(--color-primary-9, #e93d82);
}
```

## Details

The `input` placeholder text should not be selectable.

```css
.input::placeholder {
  user-select: none;
}
```
