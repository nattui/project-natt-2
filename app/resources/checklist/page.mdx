# Design checklist

This is a checklist page.

I am trying out Specification Driven Development (SDD) for creating a react component design system.

## Input

The `input` element should have a visible focus indicator.

```css
.input:focus-visible {
  outline-color: var(--color-primary-9, #e93d82);
}
```

The `input` placeholder text should not be selectable.

```css
.input::placeholder {
  user-select: none;
}
```
